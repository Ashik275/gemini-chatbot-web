# 🤖 Vanilla JS AI Chatbot

A modern and interactive chatbot web application built using **HTML, CSS, and JavaScript**, integrated with **Google Gemini API**.

## 🚀 Features

* 💬 Real-time chatbot interface
* 🤖 AI-powered responses (Gemini API)
* 😊 Emoji picker support
* 📎 Image upload with preview
* ⌨️ Auto-resizing textarea
* 🔄 Chat history handling
* 🎨 Clean and responsive UI
* ⚡ Smooth animations & typing indicator

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox + Animations)
* Vanilla JavaScript (ES6)
* Google Gemini API
* Emoji Mart (for emoji picker)



## ⚙️ Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/Ashik275/gemini-chatbot-web.git
```

2. Navigate to project folder:

```bash
cd vanilla-js-ai-chatbot
```

3. Open `script.js` and add your API key:

```js
const API_KEY = 'YOUR_API_KEY_HERE';
```

4. Run the project:

* Open `index.html` in your browser
